utils.score_functions module

Ce module fournit des fonctions pour calculer des scores liés aux évaluations des recettes.

utils.score_functions.bayesian_average(r, v, c, m=5)[source]

Calcule le score bayésien moyen pour une recette donnée.

Cette méthode prend en compte la moyenne des notes, le nombre d’évaluations, et la moyenne globale pour produire un score équilibré.

Parameters:

  • r (float) – Moyenne des notes pour une recette.

  • v (int) – Nombre d’évaluations pour cette recette.

  • c (float) – Moyenne globale des notes sur l’ensemble du dataset.

  • m (int, optional) – Seuil minimum de votes requis pour qu’un item soit considéré pertinent (par défaut 5).

Returns:

Le score bayésien moyen pour la recette.

Return type:

float
